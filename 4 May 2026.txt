3.What is a `.md` File?

* `.md` stands for **Markdown**

* It is a way to write documents that can be formatted

* It uses symbols like:

* `#` to make headings

* `*` to make lists

* It is easy to read. Can be turned into web pages 

 What to Put in Your README File

1. The title of your project

2. A description of what it does

3. Steps to install it

4. Instructions on how to use it

5. Rules for contributing to the project

6. License information
